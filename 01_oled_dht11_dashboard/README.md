## Β. Περιγραφή

Ένα απλό και κατανοητό Arduino project για αρχάριους που υλοποιεί ένα μικρό dashboard περιβάλλοντος σε OLED οθόνη.

Η εφαρμογή εμφανίζει σε πραγματικό χρόνο θερμοκρασία και υγρασία, με καθαρή διάταξη και βασικά γραφικά στοιχεία.

---

## Γ. Λειτουργίες

- Scrolling τίτλος στην πάνω μπάρα
- Θερμοκρασία (με 1 δεκαδικό)
- Υγρασία (%)
- Εικονίδια (θερμόμετρο & σταγόνα)
- Μη μπλοκαριστικός κώδικας (timers με `millis()` αντί για `delay()`)

---

## Δ. Υλικά

- Arduino UNO ή Nano
- OLED SSD1306 128x64 (I2C, διεύθυνση 0x3C)
- Αισθητήρας DHT11
- Καλώδια σύνδεσης

---

## Ε. Συνδεσμολογία

### OLED (I2C)

- VCC → 5V  
- GND → GND  
- SDA → A4  
- SCL → A5  

### DHT11

- VCC → 5V  
- GND → GND  
- DATA → D2  

---

## ΣΤ. Εκτέλεση

1. Άνοιξε το αρχείο:
oled_dht11_dashboard.ino
2. Εγκατέστησε τις απαραίτητες βιβλιοθήκες:
- Adafruit SSD1306
- Adafruit GFX
- DHT sensor library
3. Επίλεξε τη σωστή πλακέτα (UNO/Nano)
4. Upload το sketch

---

## Ε. Εκπαιδευτικοί στόχοι

- Ανάγνωση δεδομένων από αισθητήρα (DHT11)
- Εμφάνιση δεδομένων σε OLED
- Χρήση I2C επικοινωνίας
- Κατανόηση non-blocking προγραμματισμού με `millis()`
- Βασική σχεδίαση UI σε embedded σύστημα
