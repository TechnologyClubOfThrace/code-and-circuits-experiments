## Δ. Υλικά (Hardware)
Για την κατασκευή θα χρειαστείτε:
* **1x ESP Ideaskpark**
* **1x Breadboard (Optional)**
* **2x Jumper Wires (Dupont)**
* **1x Push Button**
* **🔋 Τροφοδοσία:** Μέσω USB (δυνατότητα για προσθήκη μπαταρίας 9V για φορητότητα).

## Ε. Συνδεσμολογία (Pinout)

| **GND** | Button Pin B |
