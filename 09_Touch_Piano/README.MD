## Β. Περιγραφή
Τον Νοέμβριο του 2024, μέλη του Συλλόγου Τεχνολογίας Θράκης δημιούργησαν ένα πρωτότυπο πιανάκι, προσθέτοντας μια δημιουργική νότα στην ενασχόληση με την τεχνολογία.

Το πιανάκι βασίζεται στην αλληλεπίδραση μέσω των capacitive sensors του ESP32, αναδεικνύοντας με διασκεδαστικό τρόπο τις δυνατότητες των microcontrollers και την ευρηματικότητα των μελών του συλλόγου. Διαθέτει επτά νότες, καλύπτοντας μία οκτάβα, επιτρέποντας μουσικό πειραματισμό μέσω ενός αντισυμβατικού interface. Η λειτουργία του στηρίζεται στη χωρητική ανίχνευση: ο ESP32 ανιχνεύει αλλαγές στην ηλεκτρική χωρητικότητα όταν ένα αγώγιμο αντικείμενο, όπως ένα δάχτυλο, πλησιάζει τις επιφάνειες αφής. Για τις επαφές των πλήκτρων χρησιμοποιήθηκαν αγώγιμες ταινίες, οι οποίες λειτουργούν ως πλήκτρα αφής. Αγγίζοντας μια ταινία, ενεργοποιείται η αντίστοιχη χωριτική επαφή και παράγεται ο ήχος της νότας.

Η ομορφιά του project είναι η ευελιξία: αντί για ταινίες, θα μπορούσαν ως πλήκτρα να χρησιμοποιηθούν άλλα αγώγιμα υλικά όπως τα φρούτα, λαχανικά και άλλα αντικείμενα. Αυτό το πιανάκι είναι ένα εξαιρετικό παράδειγμα του πόσο διασκεδαστική μπορεί να γίνει η εκμάθηση τεχνολογίας, προγραμματισμού και ηλεκτρονικής. Προσφέρει πρακτική κατανόηση των capacitive sensors και των διαδραστικών εφαρμογών τους, συνδυάζοντας μάθηση και ψυχαγωγία.

## Δ. Υλικά (Hardware)
Για την κατασκευή θα χρειαστείτε:
* **1x ESP32**
* **7x Jumper Wires** (Dupont)
* **7x Αγώγιμες αυτοκόλλητες ταινίες για πλήκτρα**
* **1x Ηχείο στην έξοδο 25 του ESP32**
* **🔋 Τροφοδοσία:** Μέσω USB 
* ** Έξοδος GND του ESP32  --> GND εξωδου AUX ** 
* **PIN25 του ESP32 --> AUX input pin **
