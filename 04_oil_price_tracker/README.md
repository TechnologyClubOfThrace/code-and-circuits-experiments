🔍 ΤΙ ΚΑΝΕΙ:
Συνδέεται στο WiFi, αντλεί δεδομένα μέσω API και δείχνει στην οθόνη την τιμή και την τάση (ανοδική/καθοδική τάση τιμής).

🌐 ΣΧΕΤΙΚΑ ΜΕ ΤΟ API:
Χρησιμοποιούμε ένα ανοιχτό API χωρίς Key, με περιορισμό στις κλήσει ανά μήνα.

🔋 ΤΡΟΦΟΔΟΣΙΑ:
Λειτουργεί με USB, αλλά μπορεί να γίνει πλήρως ασύρματο με προσθήκη μπαταρίας.

💻 SOFTWARE:
Χρησιμοποιεί βιβλιοθήκες όπως HTTPClient, ArduinoJSON και Adafruit. Η υποδομή είναι ευέλικτη: μπορείτε να καλέσετε οποιοδήποτε API ή και περισσότερα από ένα, εμείς απλά σας ανοίγουμε την όρεξη.

🛠️ HARDWARE:
• ESP32 C3 Supermini
• Screen OLED I2C
• 4x F2F dupont (standard jumpper wires)

🔌 ΣΥΝΔΕΣΜΟΛΟΓΙΑ (Pins):
• Pin 8: SDA
• Pin 9: SCL
• GND: GND
• 3.3V: VCC
