## Β. Περιγραφή

🔍 **Τι κάνει:**
Συνδέεται στο WiFi, αντλεί δεδομένα σε πραγματικό χρόνο μέσω API και απεικονίζει στην οθόνη την τρέχουσα τιμή του πετρελαίου Brent, καθώς και την τάση (ανοδική ή καθοδική).

---

## Γ. Λειτουργίες & Software

* **🌐 API Integration:** Χρήση ανοιχτού API (χωρίς Key) με ορίων κλήσεων ανά μήνα.
* **💻 Υποδομή:** Ευέλικτος κώδικας που επιτρέπει την κλήση πολλαπλών APIs.
* **📚 Βιβλιοθήκες:** * `HTTPClient` (για τη σύνδεση)
    * `ArduinoJSON` (για την ανάλυση δεδομένων)
    * `Adafruit GFX & SSD1306` (για την απεικόνιση)

---

## Δ. Υλικά (Hardware)

* **ESP32 C3 Supermini**
* **OLED Screen I2C** (128x64)
* **4x F2F Dupont wires** (standard jumper wires)
* **🔋 Τροφοδοσία:** Μέσω USB (δυνατότητα για προσθήκη μπαταρίας για ασύρματη λειτουργία).

---

## Ε. Συνδεσμολογία (Pinout)

| OLED Pin | ESP32 C3 Pin | Περιγραφή |
| :--- | :--- | :--- |
| **VCC** | 3.3V | Τροφοδοσία |
| **GND** | GND | Γείωση |
| **SDA** | Pin 8 | Δεδομένα I2C |
| **SCL** | Pin 9 | Ρολόι I2C |

## Ζ. Σημειώσεις

Το project αποτελεί μια εξαιρετική βάση για την εκμάθηση διαχείρισης JSON δεδομένων και Web Requests σε embedded συστήματα. Η υποδομή είναι έτοιμη για να την "κουμπώσετε" οποιοδήποτε άλλο API σας ενδιαφέρει.
