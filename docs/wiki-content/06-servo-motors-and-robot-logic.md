# Τι είναι το servo motor και πώς “σκέφτεται” ένα robot;

Στα robot projects συναντάμε συχνά λέξεις όπως `servo`, `state machine`, `mode`, `obstacle avoidance`, `line tracking`. Όλα αυτά μπορεί να ακούγονται προχωρημένα, αλλά στην ουσία περιγράφουν πολύ λογικές και κατανοητές ιδέες.

## Τι είναι servo motor;

Το servo είναι ένα μοτεράκι που δεν γυρίζει απλώς ελεύθερα. Μπορεί να τοποθετηθεί σε συγκεκριμένη γωνία. Αυτό το κάνει πολύ χρήσιμο όταν θέλουμε ελεγχόμενη κίνηση, όπως:

- κλίση πλατφόρμας
- στροφή αισθητήρα
- κίνηση κεφαλιού

Για αυτό βλέπουμε servo στο Gravity Maze, στο robot car και στο R2inoD2ino.

## Τι διαφορά έχει από ένα απλό DC motor;

Ένα απλό DC motor είναι καλύτερο όταν θέλουμε συνεχή περιστροφή, όπως στους τροχούς ενός ρομπότ. Ένα servo είναι καλύτερο όταν θέλουμε ελεγχόμενη θέση.

## Τι είναι το line tracking;

Είναι η ικανότητα ενός ρομπότ να ακολουθεί μια γραμμή στο πάτωμα, συνήθως με ειδικούς αισθητήρες. Το ρομπότ διαβάζει τι “βλέπουν” οι αισθητήρες και διορθώνει την πορεία του.

## Τι είναι το obstacle avoidance;

Είναι η αποφυγή εμποδίων. Δηλαδή το ρομπότ μετρά αποστάσεις και αποφασίζει προς τα πού να κινηθεί χωρίς να χτυπήσει.

## Τι είναι `state machine`;

Είναι ένας πολύ οργανωμένος τρόπος να γράφουμε συμπεριφορά. Αντί να έχουμε έναν κώδικα που μπερδεύεται εύκολα, χωρίζουμε τη λειτουργία σε “καταστάσεις”.

Για παράδειγμα, ένα robot μπορεί να είναι σε:

- `MANUAL`
- `LINE TRACKING`
- `OBSTACLE AVOIDANCE`
- `STOP`

Αυτό βοηθά πολύ, γιατί το project γίνεται πιο καθαρό, πιο αξιόπιστο και πιο εύκολο στη βελτίωση.

## Τι σημαίνει fail-safe;

Σημαίνει ότι το project έχει προβλέψει τι θα κάνει όταν κάτι πάει στραβά. Για παράδειγμα:

- αν δεν λάβει σωστά δεδομένα
- αν κολλήσει
- αν δεν βρίσκει ασφαλή διαδρομή

Σε εκπαιδευτικά projects αυτό είναι πολύ σημαντικό, γιατί δεν θέλουμε μια κατασκευή να κινείται απρόβλεπτα.

## Τι να κρατήσει ένας αρχάριος;

Το πιο βασικό που αξίζει να κρατήσει ένας αρχάριος είναι ότι τα robot projects δεν είναι “μαγικά”. Συνήθως κάνουν τρία πράγματα:

1. Διαβάζουν αισθητήρες.
2. Παίρνουν αποφάσεις με βάση κανόνες.
3. Στέλνουν εντολές σε motors ή servos.

Αυτό είναι, σε γενικές γραμμές, όλο το βασικό τους νόημα.
