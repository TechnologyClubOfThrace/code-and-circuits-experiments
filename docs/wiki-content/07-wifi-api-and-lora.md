# Τι είναι το Wi-Fi, το API και το LoRa σε ένα project;

Μέσα στα projects του repository βλέπουμε δύο πολύ ενδιαφέροντες δρόμους επικοινωνίας:

- επικοινωνία μέσω internet
- ασύρματη επικοινωνία μεγάλων αποστάσεων

Αυτοί οι δύο κόσμοι μοιάζουν μεταξύ τους, αλλά στην πράξη εξυπηρετούν διαφορετικές ανάγκες.

## Τι είναι το Wi-Fi σε ένα project;

Όταν ένα ESP32 συνδέεται στο Wi-Fi, μπορεί να επικοινωνεί με υπηρεσίες στο internet. Αυτό επιτρέπει σε μια κατασκευή να παίρνει δεδομένα από ιστοσελίδες ή από online υπηρεσίες.

Στο oil tracker, για παράδειγμα, η πλακέτα συνδέεται στο internet και ζητά την τρέχουσα τιμή του πετρελαίου Brent.

## Τι είναι API;

Ένα `API` είναι ένας οργανωμένος τρόπος με τον οποίο μια υπηρεσία μάς δίνει δεδομένα. Αντί να “διαβάζουμε” μια ολόκληρη ιστοσελίδα, ζητάμε απευθείας τις πληροφορίες που θέλουμε.

Έτσι το project μπορεί να πάρει:

- τιμές
- μετρήσεις
- καταστάσεις
- απαντήσεις σε μορφή `JSON`

## Τι είναι JSON;

Το `JSON` είναι ένας τρόπος οργάνωσης δεδομένων. Μοιάζει με ένα τακτοποιημένο πακέτο πληροφοριών που ο κώδικας μπορεί να διαβάσει και να χρησιμοποιήσει.

## Και το LoRa τι είναι;

Το `LoRa` είναι μια τεχνολογία ασύρματης επικοινωνίας που είναι σχεδιασμένη για μεγάλες αποστάσεις και χαμηλή κατανάλωση. Δεν είναι internet και δεν είναι Wi-Fi. Είναι κάτι διαφορετικό.

Στο LoRa project του repository, έχουμε:

- έναν transmitter που στέλνει θερμοκρασία
- έναν receiver που την λαμβάνει

Αυτό βοηθά έναν αρχάριο να καταλάβει πολύ καθαρά την έννοια του πομπού και του δέκτη.

## Ποια είναι η βασική διαφορά Wi-Fi και LoRa;

Με πολύ απλά λόγια:

- το Wi-Fi χρησιμοποιείται όταν θέλουμε πρόσβαση σε δίκτυο και internet
- το LoRa χρησιμοποιείται όταν θέλουμε ασύρματη επικοινωνία σε απόσταση, χωρίς να βασιζόμαστε στο internet

## Τι σημαίνει RSSI;

Το `RSSI` είναι μια ένδειξη ισχύος του σήματος. Δεν είναι τέλειο μέτρο, αλλά μας βοηθά να έχουμε μια εικόνα για το πόσο καλά λαμβάνεται ένα σήμα.

## Τι να κρατήσει ένας αρχάριος;

Δεν χρειάζεται να καταλάβει όλη τη θεωρία των δικτύων. Για αρχή αρκεί να ξεχωρίσει:

- άλλο πράγμα είναι να ζητάω δεδομένα από το internet
- άλλο πράγμα είναι να στέλνω δεδομένα ασύρματα από μια συσκευή σε μια άλλη

Αυτή η διάκριση αρκεί για να αρχίσει να βγάζει νόημα ο κόσμος των connected projects.
