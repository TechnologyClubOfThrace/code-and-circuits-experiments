# Wiki Content for GitHub

Αυτός ο φάκελος περιέχει έτοιμα κείμενα για το GitHub Wiki του repository, γραμμένα σε απλή, κατανοητή γλώσσα για αρχάριους.

Η ιδέα είναι απλή:

- Κάθε αρχείο Markdown αντιστοιχεί σε μία πιθανή σελίδα του wiki.
- Το ύφος είναι φιλικό και επεξηγηματικό, χωρίς περιττή τεχνική ορολογία.
- Τα παραδείγματα πατάνε πάνω στα projects του repository.

## Σημείωση

Τα κείμενα είναι γραμμένα ώστε να διαβάζονται εύκολα και αυτόνομα. Αν θέλεις, μπορείς αργότερα να τα εμπλουτίσεις με εικόνες, συνδέσμους προς project folders και μικρά παραδείγματα κώδικα.
