09. Touch Piano με ESP32

Μουσικό project με capacitive touch πλήκτρα, όπου το ESP32 ανιχνεύει την αφή και παίζει επτά νότες μέσα από ένα απλό, δημιουργικό interface.

Περιγραφή

Το Touch Piano αξιοποιεί τις χωρητικές εισόδους του ESP32 ώστε οι νότες να ενεργοποιούνται με άγγιγμα σε αγώγιμες ταινίες. Με αυτόν τον τρόπο, το project συνδυάζει ηλεκτρονική, ήχο και αλληλεπίδραση σε μία πολύ άμεση κατασκευή.

Ολοκλήρωση της κατασκευής στον Σύλλογο Τεχνολογίας Θράκης

Ομάδα Κατασκευής: Μπορκέλης Γ., Γιάννης Γ., Άρης Τ.

Α. Λειτουργίες λογισμικού

  • Ανίχνευση αφής μέσω capacitive touch αισθητήρων του ESP32
  • Αναπαραγωγή 7 νοτών που καλύπτουν μία οκτάβα
  • Χρήση αγώγιμων επιφανειών ως πλήκτρα με πολύ απλή κατασκευή
  • Εύκολη προσαρμογή για άλλα αγώγιμα αντικείμενα ή διαδραστικά interfaces

Β. Υλικά

  • 1x ESP32
  • 7x Jumper wires
  • 7x Αγώγιμες αυτοκόλλητες ταινίες για πλήκτρα αφής
  • 1x Ηχείο ή έξοδος ήχου συνδεδεμένη στο pin 25
  • Τροφοδοσία μέσω USB

Γ. Συνδεσμολογία

  • T0 (G4) → Νότα do
  • T3 (G15) → Νότα re
  • T4 (G13) → Νότα mi
  • T5 (G12) → Νότα fa
  • T6 (G14) → Νότα sol
  • T7 (G27) → Νότα la
  • T8 (G33) → Νότα si
  • GND → Γείωση εξόδου ήχου
  • PIN25 → AUX input / έξοδος ήχου

Δ. Εκπαιδευτική αξία

Το project δείχνει πολύ καθαρά πώς μπορούμε να φτιάξουμε διαδραστικές εφαρμογές με capacitive sensing, μετατρέποντας απλά υλικά σε μουσικά πλήκτρα. Είναι ιδανικό για εισαγωγή σε αισθητήρες αφής, φυσικό computing και δημιουργικό προγραμματισμό.
